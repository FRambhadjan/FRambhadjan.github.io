Doelstelling



Mijn doel met deze website is het te laten zien aan toekomstige werkgevers om mijn werkgevers te laten zien wat ik kan en wat ik bezit.
Ik probeer zo veel mogelijk professionele keuzes te maken voor mijn website en een professionele huisstijl.
Voor de opdracht moest ik ook profesionele kleuren en huisstijl gebruiken. Toen was ik gaan spelen met hoe ik de website eruit zou laten zien,
ik kon kiezen tussen een website die heel complex is of een simpele website. Ik heb gekozen om een
simpele website te bouwen want bedrijven tegenwoordig maken alles zo simpel mogelijk.

Ik wil deze site ook gebruiken voor meerdere portfolios en zo veel mogelijk verbeteren na elke design periode.
Deze porfolio kan ook misschien later een bouwsteen worden voor een echte website die ik kan gaan maken.
Op dit moment voelt de website wel goed maar ik weet zeker dat ik de website beter kan krijgen en maken.
